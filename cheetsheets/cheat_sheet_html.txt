Расшифровка сокращенных имен тэгов и атрибутов:
    p           paragraph
    h1 - h6     headings
    b           bold
    em          emphasis
    i           italic
    a           anchor
    img         image
    br          line break
    hr          horizontal rule
    ol          ordered list
    ul          unordered list
    li          list item
    dl          definition list
    dt          definition term
    dd          definition description
    tr          table row
    th          table header cell
    td          table data cell
    div         division
    iframe      inline frame
    nav         navigation

    href        hypertext reference
    src         source
    colspan     how many columns the cell extends
    rowspan     how many rows the cell extends
    charset     character encoding
    rel         relationship of the linked document to the current document

Спецсимволы HTML (HTML-код + CSS-код)
    &nbsp;      \00A0       Неразрывный пробел
    &lt;        \003C       Знак "меньше чем" (начало тега)
    &gt;        \003E       Знак "больше чем" (конец тега)
    &copy;      \00A9       Знак copyright
    &reg;       \00AE       Знак зарегистрированной торговой марки
    &trade;     \2122       Знак торговой марки
