jQuery API https://api.jquery.com/

Ajax
    jQuery.ajax() Выполняет асинхронный HTTP (Ajax) запрос.
    .serialize() Возвращает строку, с именами и значениями выбранных элементов формы. Эта строка будет иметь формат параметров url-запроса.
    .serializeArray() Упорядочивает все формы и элементы на форме (подобно .serialize()), но возвращает данные, которые можно использовать в структуре JSON

Attributes
    .addClass() Добавляет класс(ы) выбранным элементам страницы. Если из этих элементов некоторые уже принадлежат каким-то классам, то новый (новые) класс добавится к существующим, а не заменит их.
    .attr() Возвращает или изменяет значение атрибутов у выбранных элементов страницы. Функция имеет четыре варианта использования.
    .hasClass() Проверяет наличие класса у элементов страницы.
    .html() Возвращает или изменяет html-содержимое выбранных элементов страницы.
    .removeAttr() Метод для удаления атрибутов (таких как id, class, title и.т.д) у элементов страницы.
    .removeClass() Удаляет заданные классы у элементов на странице.
    .toggleClass() Добавляет или удаляет заданный класс(ы) по принципу переключателя (добавляет, если элемент не содержит класса, и удаляет, если класс есть).
    .val() Метод позволяет получать и изменять значения элементов форм. Для элементов input это значение атрибута value; для списков выбора (select) – значение value выбранного элемента (в случае множественного выбора – массив значений); в случае с textarea, метод .val() будет работать непосредственно с содержимым тега textarea.

Core
    jQuery() - $() - Функция jQuery() является, пожалуй основной функцией в библиотеке. Она обычно, вызывается намного чаще других и поэтому для нее существует короткая форма записи — $(). Функция jQuery() позволяет производить поиск элементов на странице, создавать новые элементы по заданному HTML тексту, а так же кое что еще. Возвращает объект jQuery.
    jQuery.noConflict() Відмовтеся від контролю jQuery над змінною $.

CSS
    .css() Возвращает или изменяет значения css-величин у выбранных элементов страницы.

Data
    .data() С помощью этого метода, появляется возможность привязывать собственные переменные к любым элементам страницы.

Events
    .click() Устанавливает обработчик "клика" мышью по элементу, либо, запускает это событие.
    .off() Удаляет с выбранных элементов страницы обработчики событий, установленные с помощью метода .on().
    .on() Устанавливает обработчики событий на выбранные элементы страницы.
    .one() Устанавливает обработчик события выбранным элементам страницы. Особенностью метода является то, что обработчик будет вызван не более одного раза, на каждом из элементов.
    .trigger() Вызывает событие у выбранных элементов, что приводит к запуску обработчиков этого события.

Manipulation
    .after() Функция вставляет заданное содержимое сразу после определенных элементов страницы.
    .append() Вставляет содержимое, заданное параметром, в конец каждого элемента в наборе соответствующих элементов.
    .appendTo() Функция добавляет содержимое в конец элементов.
    .before() Функция помещает заданное содержимое перед определенными элементами страницы.
    .clone() Метод создает копии выбранных элементов страницы и возвращает их в виде объекта jQuery. Элементы копируются вместе со всеми содержащимися внутри них элементами.
    .empty() Удаляет все содержимое у выбранных элементов (без удаления самих элементов).
    .insertAfter() Функция вставляет заданное содержимое сразу после определенных элементов страницы.
    .insertBefore() Функция помещает заданное содержимое перед определенными элементами страницы.
    .prepend() Функция добавляет содержимое в начало определенных элементов.
    .prependTo() Функция добавляет все выбранные элементы внутрь другого набора элементов перед существующим контентом.
    .remove() Метод для удаления элементов страницы.
    .replaceAll() С помощью этой функции можно заменять элементы страницы новыми элементами или уже существующими.
    .replaceWith()С помощью этой функции можно заменять элементы страницы новыми элементами или уже существующими.
    .text() Возвращает или изменяет текстовое содержимое выбранных элементов страницы.
    .unwrap() Удаляет родительские элементы у выбранных элементов, при этом, их содержимое останется на своих местах.
    .wrap() Функция помещает заданное содержимое вокруг выбранных элементов (как бы "обертывая" их).
    .wrapInner() "Обертывает" содержимое выбранных элементов заданными html-элементами. Таким образом получается, что все выбранные объекты "обертываются изнутри".

Miscellaneous
    .each() Итерация над объектом JQuery, выполняет функцию для каждого элемента.
    .get() Осуществляет запрос к серверу методом GET, без перезагрузки страницы.
    .index() Возвращает индекс заданного элемента в наборе (2 и 3 вариант использования) или относительно соседних элементов (1 вариант использования).
    .toArray() Возвращает массив всех DOM-элементов, хранящихся в объекте jQuery.

Properties
    .length Содержит количество выбранных элементов.

Traversing
    .add() Добавляет заданные элементы в уже существующий набор элементов.
    .children() Находит дочерние элементы.
    .closest() Для каждого элемента в наборе, получает первый элемент, который совпадает с селектором при движении вверху по DOM дереву элементов.
    .eq() Возвращает элемент, идущий под заданным номером в наборе выбранных элементов.
    .find() Поиск потомков внутри каждого элемента в текущем наборе соотвествующих элементов с фильтрацией по селектору, объекту jQuery или элементу.
    .has() Фильтрует набор выбранных элементов, оставляя только те, которые имеют определенных потомков.
    .first()Возвращает первый элемент из всех выбранных. Эквивалентно .eq(0).
    .last() Возвращает последний элемент из всех выбранных. Эквивалентно .eq(-1).
    .next() Осуществляет поиск элементов, лежащих непосредственно после заданных элементов (по одному для каждого из заданных).
    .nextAll() Осуществляет поиск всех элементов, лежащих после заданных элементов.
    .parent() Осуществляет поиск родительских элементов всех заданных элементов.
    .prev() Для каждого из выбранных элементов находит предшествующий ему элемент (но только если он лежит на том же уровне иерархии DOM).
    .prevAll() Осуществляет поиск всех элементов, лежащих перед заданными элементами.
 
Utilities
    jQuery.contains() Проверяет, является ли элемент DOM потомком другого DOM элемента.
